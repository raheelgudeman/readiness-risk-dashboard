# Readiness Risk Dashboard

## Overview
This project analyzes operational readiness data to identify personnel, vehicle, and training risks across units.  
The goal is to support leadership decision-making using data-driven risk indicators, visualizations, and scoring models.

## Objectives
- Quantify readiness risks across personnel, vehicles, and training
- Identify units at highest operational risk
- Build a composite readiness risk score
- Visualize readiness trends and vulnerabilities
- Lay groundwork for predictive risk modeling

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook
